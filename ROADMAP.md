## 🛠️ Технические примечания

### Архитектурные изменения
- **Конвертеры**: Создать базовый класс `BaseConverter` для разных форматов
- **UI**: Добавить выпадающий список для выбора формата вывода
- **Config**: Вынести расширения файлов в настраиваемый список

### Зависимости
- **HTML/PDF**: `weasyprint` или `reportlab`
- **Подсветка синтаксиса**: `pygments` (уже может быть установлен)
- **Drag & Drop**: Встроено в PyQt6

### Тестирование
- Создать тестовые файлы для каждого поддерживаемого языка
- Проверка корректности конвертации для разных форматов
- Тестирование производительности на средних проектах (50-200 файлов)
