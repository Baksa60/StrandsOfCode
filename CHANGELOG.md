# Changelog StrandsOfCode

Все заметные изменения проекта будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.3.0] - 2026-03-19

### ✨ Добавлено
- 🎯 **Drag & Drop функциональность** - перетаскивание файлов и папок в окно приложения
- 🎨 **Визуальная индикация зоны Drop** - зеленая рамка при наведении файлов
- 🧠 **Умное определение формата файлов** (90-95% точность)
- ⚙️ **Автоматический выбор типа источника** на основе содержимого
- 📁 **Умное определение типа папки** (простая/рекурсивная)
- 🔄 **Поддержка множественного drag & drop**
- 📋 **Обработка разных типов контента** (файлы, папки, смешанное)
- 📦 **Система версионирования продукта**
- 📄 **Обновленная документация** с полной архитектурой проекта

### 🚀 Улучшено
- Алгоритм определения формата файлов с учетом доминирования
- Логика выбора типа источника при перетаскивании
- Визуальная обратная связь при Drag & Drop
- Структура документации и метаданных

### 🐛 Исправлено
- Проблемы с определением формата в папках
- Неправильная обработка смешанного контента
- Ошибки в подсчете файлов для определения доминирования
- Убраны отладочные логи из production

## [1.1.0] - 2026-03-17

### ✨ Добавлено
- 💾 **Умная система сохранения** с выбором папки ДО конвертации
- 📝 **Генерация уникальных имен файлов** с временными метками
- 💾 **Запоминание настроек** между сессиями
- 🎲 **Кнопка генерации имени файла**
- 📂 **Выбор места сохранения** с предварительным выбором
- 📄 **Поддержка Markdown формата** с подсветкой синтаксиса
- 📊 **Поддержка JSON формата** для структурированных данных
- 🔄 **Обратная конвертация** из TXT/MD/JSON в код

### 🚀 Улучшено
- Пользовательский опыт сохранения файлов
- Предотвращение перезаписи существующих файлов
- Удобство выбора папки назначения

### 🐛 Исправлено
- Проблемы с правами доступа при сохранении
- Лишняя вложенность папок (матрешка)
