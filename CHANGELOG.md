# Changelog StrandsOfCode

Все заметные изменения проекта будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.1.0] - 2026-03-18

### ✨ Добавлено
- **Поддержка JavaScript файлов (.js, .jsx)**
- **Поддержка TypeScript файлов (.ts, .tsx)**
- **Автоопределение JavaScript/TypeScript проектов**
- **Кнопка "О программе" с подробной информацией**
- **Тёмная тема по умолчанию**
- **Уникальное название StrandsOfCode** с метафорой прядения кода

### 🐛 Исправлено
- Удалены предупреждения "Unknown property transform" в консоли
- Исправлена совместимость с PyQt6 (Qt.AlignmentFlag)

### 🔄 Изменено
- **Полный ребрендинг проекта** из py2txt_tool в StrandsOfCode
- Обновлен заголовок приложения с "Python конвертер" на "Конвертер кода"
- Удален подзаголовок из главного окна
- Расширены фильтры файлов в диалогах выбора

### 🧬 Философия
- Внедрена уникальная метафора прядения нитей кода
- Обновлена концепция бренда и документация

### 📦 Техническое
- Добавлен `.gitignore` для публикации на GitHub
- Создана документация проекта с новой философией
- Подготовлена структура для релизов
- Обновлены все файлы проекта с новым названием

## [1.0.0] - 2026-03-17

### ✨ Добавлено
- **Базовая функциональность конвертера**
- **Поддержка Python файлов (.py)**
- **Определение Python проектов**
- **Современный интерфейс на PyQt6**
- **Светлая и тёмная темы**
- **Режимы конвертации: отдельные файлы / объединенный**
- **Статистика проектов и прогресс**
- **История недавних файлов**
- **Метаданные и нумерация строк**
- **Лог операций**

### 🏗️ Архитектура
- MVC паттерн
- Модульная структура
- Конфигурируемые параметры
- Расширяемая архитектура конвертеров
